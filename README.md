# Kingpin

Google makes it deliberately hard to export your data. This tool tries to help
you get your pins out of Google without losing context about them.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install:

```bash
pip install .
```

## Usage

Run kingpin in your favorite shell:

```bash
# Load pins from "saved places" csv
kingpin load favorite.csv

# List loaded pins
kingpin list
```
